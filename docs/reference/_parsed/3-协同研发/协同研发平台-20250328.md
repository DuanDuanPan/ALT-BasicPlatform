协同发起人不了解整体进展，进度靠线下逐层线下电话询问，进展获取与跟催效率低

## Slide 44

技术架构 



**Notes:**
我们平台采用了微服务的架构、支持国产化的数据库和大数据环境。
我们系统采用了微服务的技术架构，并在国产化方面，可支持从cpu、操作系统、数据库的全国产化。
本架构的主要亮点：
执行性能：针对某些高频使用的模块可以部署多个实例来提高执行效率。
可靠性、稳定性：通过多个实例的部署，保证系统始终可用。
弹性，扩展性：各模块间松耦合，数据与应用分离，后期平台扩展功能时，可以扩展新的微服务来实现而不需要更改已经完成的服务。
安全性：系统采用统一的网关服务，任何访问请求必须采用经过安全认证。
可运维性：整个主数据平台构建了统一的注册中心和配置中心，运维时可以进行统一配置和系统运行情况监控。
