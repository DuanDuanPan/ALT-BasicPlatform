杰为公司简介
   成立于2007年，总部位于深圳市南山区高新科技园，在
北京、武汉、顺德和中国香港等地设有分支机构。
   “用互联网技术重构工业软件”，致力为工业企业数字化转
型赋能，帮助工业企业升级和替换传统工业软件。
    深度融合了边缘计算、云计算、人工智能、RPA、图型
数据库，低代码开发、MASA、DevOps等技术,先后研发了
全套JWI Digital系列工业软件产品。
      已成功应用于国防军工、汽车交通运输、工业机械、高科电

全生命周期管理的解决方案。可进行实时数
据采集与控制，实现设备预防性维护和智能
故障分析，改进资产绩效、降低成本、增加
盈利能力并减少项目风险。

JWI QMS

JWI QMS (质量管理) 是工厂制造全过程的
数字化质量管理软件。实现制造过程中关键工
序、器件、指标的实时采集、监控、智能分析
和预警，解决产品质量事件的快速响应和闭环
管理，为产品质量持续改善提供决策依据。

JWI MDE

JWI MDE (制造调度引擎) 是一款以任务链
为主线的制造调度软件。它以制造任务链为
主线，将工艺数据和制造资源数据、物流资
源数据联系起来，对工艺、计划、物管、仓
储配送、生产制造等各部门数据的调度和控
制，确保数据的一致性，构建从工艺规划到
制造执行的数字制造链。
