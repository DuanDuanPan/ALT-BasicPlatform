www.peracd.com
PERA Chengdu Inc. 
产品定位
多学科协同仿真平台（英文名：SDMan）是成都安世亚太科技有限公司自主研发的产品，针对
多学科多专业的计算机仿真数据进行结构化管理、对仿真流程进行规范化管理、对仿真工具进行
集成管理，实现多学科仿真过程的协同管理。
SDMan可解决仿真过程中的如下管理问题：
1）海量仿真分析数据缺乏有效的管理
2）仿真流程和仿真经验不能有效地共享
3）多学科仿真过程无法协同和有效监控
4）各种商用和自研仿真工具缺乏有效的集成
