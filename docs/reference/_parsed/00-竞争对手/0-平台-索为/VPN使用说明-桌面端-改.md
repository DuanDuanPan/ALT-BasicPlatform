# VPN使用说明-桌面端-改

VPN开通账号之后会往企业邮箱发送密码，第一次登录后需要更改密码

在电脑端打开浏览器输入下面的地址

https://60.247.58.139:55555/login/login.html  登录后点击下载，根据自己电脑所属平台下载对应客户端 

界面如下：












下载完毕后打开客户端点击红框中的齿轮








点击齿轮后点击场景管理点击我的新场景+号弹出SSL vpn新建连接向导, 



点击下一步，网关按照下图的填写，用户名密码为已开通vpn账号密码，其他属性和图中一致即可，






上图配置完毕后，点击完成，之前编辑的场景变为绿色，点击保存即可



返回iNode客户端，点击我的新场景，点击连接即可，连接成功如下图所示，不使用时点击断开。
