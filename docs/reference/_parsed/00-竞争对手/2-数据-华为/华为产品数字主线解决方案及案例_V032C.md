## Page 5

数字主线概念：美国空军研究实验室和NASA定义的Digital Thread
• 【美国空军研究实验室和NASA 】提
出Digital Thread ：利用先进建模和
仿真工具构建的，覆盖产品全生命周
期与全价值链，从基础材料、设计、
工艺、制造以及使用维护全部环节，
集成并驱动以统一的模型为核心的产
品设计、制造和保障的数字化数据流
• 数字主线的价值：
• 在数字主线中，统一的全数字化模
型贯穿始终。所有的环节都具备信
息完整丰富、按照统一的开放标准、
建立的规范和语义化的数字化模型，
并且可被机器（或系统）稳定无歧
义地读取
• 统一数据源，产品有关的数字化模
型采用标准开放的描述，可以逐级
向下传递而不失真，也可以回溯
• 通过加大仿真系统对现实的模拟，
减少物理样机生产和实验实践。

## Page 6

业界数字主线定义总结
•
数字主线不是一个具体的系统也不是一个产品，尽管实现的本质是系列数字技术的组合，但同时其本身又
承担着其他数字技术穿针引线的使命，支撑实现由点到线、由线到面的全面数字化转型。拿人体来比喻智
能化和数字化的话，数字主线更像人体的血管和神经网络，供养着各器官，同时又智能地连接了大脑和器
官，将类似传感器收集的器官数据实时汇集到大脑，经过大脑评估后又将反馈结果指令下达给各器官执行。
•
数字主线不能仅仅在技术层面进行理解，其带来的价值将直接反应在企业生产经营的诸多方面，是打通企
业全价值链的数字化手段，并且与传统总线技术不同的是内嵌了各种产品数字化模型，而不仅仅是数据和
应用集成的信息通道，从而具有更强的智能、柔性和可扩展等特点；如果说Digital Twin等数字技术是把物
理世界中某一个或几个实物搬进了虚拟世界割裂存在，那么Digital Thread是把虚拟世界中所有的虚拟实体
连接在一起，使其成为真正的“数字社会”。
