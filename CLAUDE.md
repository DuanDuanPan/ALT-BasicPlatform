## Project Overview

This is a **documentation and architecture planning repository** for the ALT-BasicPlatform (基础底座) project — an industrial R&D platform ("工业研发底座") developed by OLTran (奥蓝托). It is **not a source code repository**; it contains architecture blueprints, planning artifacts, competitive research, reference documents, and BMAD workflow configurations.

The platform aims to unify three business lines under a shared platform kernel:
- **OLTran.CDM** — Collaborative R&D (协同研发)
- **OLTran.TDM** — Digital Testing / Test Data Management (数字化试验), built on ASAM ODS / TODS standards
- **OLTran.SUN.BPM** — Management Information Systems (管理信息化)

## Key Architecture Concepts

The architecture blueprint (`v0.5`) defines:
- **7-layer logical architecture** (L1–L7) with **4 cross-cutting planes** (security, audit, traceability, governance)
- **Unified Object Model** with `TraceLink` as a first-class citizen for end-to-end traceability
- **Multi-store strategy**: primary store writes + projection updates (no business-layer dual-writes)
- **Graph Service**: logically unified but physically separated (model graph, instance graph, knowledge graph)
- **ITP (Industrial Template Package)**: the replication and delivery unit for cross-customer deployments
- **ADR tracking**: decisions are tracked via ADR-001 through ADR-023+

## Running the Document Converter

```bash
cd docs/reference
pip install PyMuPDF python-pptx docx2txt pandas Pillow pytesseract
python scripts/local_converter.py
```

Requires `antiword` (for .doc files) and `tesseract` with `chi_sim+eng` language packs (for OCR fallback).
